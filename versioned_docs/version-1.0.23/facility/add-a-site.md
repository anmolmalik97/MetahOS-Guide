# How to add a site?

## Step 1: Hover on Facility option.

#### Facility>Site Management

![enter image description here](https://res.cloudinary.com/teleopdassets/image/upload/v1642402187/service/Site/site_1_ertmpg.png)

## Step 2: Click on Site Management then, click on Site button to add Site Code, Site Name, GST In, Address of the Site and even add the Notes on that Site. After filling all the details click on Save button.

![enter image description here](https://res.cloudinary.com/teleopdassets/image/upload/v1642402187/service/Site/site3_vqgbxo.png)

![enter image description here](https://res.cloudinary.com/teleopdassets/image/upload/v1642402188/service/Site/site4_snnyrx.png)

## Step 3: Click on Buildings to add Building code, Building Name, select the Site. And even add the Notes on that building. After filling all the details click on Save button.

![enter image description here](https://res.cloudinary.com/teleopdassets/image/upload/v1642407272/service/Site/site_building_l2l3jo.png)

![enter image description here](https://res.cloudinary.com/teleopdassets/image/upload/v1642402188/service/Site/site_6_nklere.png)

## Step 4: Click on Floors to add Floor code, floor name, and select the Building. After filling all the details click on Save button.

![enter image description here](https://res.cloudinary.com/teleopdassets/image/upload/v1642407362/service/Site/site_floor_a2pwtk.png)

![enter image description here](https://res.cloudinary.com/teleopdassets/image/upload/v1642402188/service/Site/site7_tk8emt.png)

## Step 5: Click on Rooms to add Room code, Room name, select the Floor. After filling all the details click on Save button.

![enter image description here](https://res.cloudinary.com/teleopdassets/image/upload/v1642407796/service/Site/site_room_jjz0wm.png)

![enter image description here](https://res.cloudinary.com/teleopdassets/image/upload/v1642402188/service/Site/site8_nikhzt.png)

## Step 6: Click on Shelves to add Shelf code, select the Room. After filling all the details click on Save button. Mainly Shelves will be in Pharmacy.

![enter image description here](https://res.cloudinary.com/teleopdassets/image/upload/v1642408334/service/Site/site_shelf_pgwpik.png)

![enter image description here](https://res.cloudinary.com/teleopdassets/image/upload/v1642402188/service/Site/site9_apvhuo.png)

## Step 7: Click on Racks to add Rack code, select the Shelf. After filling all the details click on Save button. Mainly Racks will be in Pharmacy.
