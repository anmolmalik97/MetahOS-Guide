SITE MANAGEMENT

Administration and oversight of various locations or facilities that are part of the healthcare network. This encompasses a range of activities and functions to ensure that the HIS operates smoothly across different sites, which may include hospitals, clinics, laboratories, and other healthcare facilities.

Click on the Facility module and choose Site Management.

After selecting Site Management, the home screen will appear.

The system includes various components such as Sites, Buildings, Floors, Rooms, Shelves, and Racks. Here’s a detailed description:

  • Sites: These are the primary locations or areas managed within the system, representing different geographical or organizational locations.
  • Buildings: Each site can contain multiple buildings, representing separate structures or facilities within the site.
  • Floors: Buildings are divided into floors, allowing for a hierarchical organization of space within each building.
  • Rooms: Each floor comprises several rooms, which are individual enclosed spaces within the floor.
  • Shelves: Rooms contain shelves, which are units used to store items in an organized manner.
  • Racks: Shelves are further divided into racks, providing additional subdivisions for storage and organization of materials.

FOR THE SITE OPTION

Select Sites and create a New Site.

Select Sites and create a new site by entering details such as the site code, name, GST In, address, location, and any relevant notes.

Click Save.

After creating a new site, it will appear on the home screen where you can perform actions such as editing or disabling it.

The Edit option allows you to make changes to the newly created site, while the Disable option lets you deactivate the site.

The site can be edited to update the information that was previously added.

FOR THE BUILDINGS OPTION

FOR FLOORS

Choosing “Floors” will display the home screen in a log format.

You can add a new floor and utilize the filter option to select specific floors.

Choosing “Select Building” facilitates filtering to find the desired type.

The edit option facilitates inputting new information.

When editing floor details, you can modify information such as floor code, name, building, and notes.

FOR ROOMS

Selecting “Rooms” will bring up the home screen for rooms.

The edit option allows for modifying existing information, while scheduling will enable the arrangement of room bookings.

Selecting the edit option provides access to modify details such as room code, name, floor, type, accessories, and notes.

The schedule feature provides information about planned appointments, displayed by day, week, and month, along with an agenda view.

The filter helps identify the required floor and displays the results for that floor only.

Click “New Room” and enter details such as room code, name, floor, type, and notes.
