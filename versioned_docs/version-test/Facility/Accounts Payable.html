ACCOUNTS PAYABLE

Proceed to the facility and select “Accounts Payable”.

All accounts payable orders will be displayed in list format, accompanied by available actions.

The reset, export CSV, and recurring order functionalities can be utilized to streamline operations on the page.

Reset - Restore to original settings or condition.

Export CSV - Save data in Comma Separated Values format for use in spreadsheets or databases.

Recurring order - A repeating purchase arrangement for goods or services at specified intervals.

  1. Refresh Button: The button with a circular arrow icon on the far left is typically used to refresh the data or apply the selected filters.

  2. Store Filter Dropdown: This dropdown menu allows you to select a specific store from a list.

  3. Purchase or Work Order Dropdown: This dropdown menu allows for filtering based on purchase orders or work orders.

  4. Approval Status Dropdown: This dropdown menu lets you filter data by approval status, such as Approved, Rejected, or Pending.

  5. Date Pickers: These fields with the format “dd-mm-yyyy” allow you to select a start date and an end date for filtering data within a specific date range. The calendar icons next to each field suggest that clicking them will open a calendar widget for easy date selection.

  1. Green Checkmark Button: This button, represented by a green checkmark, typically signifies an “Approve” action. Clicking it would approve a selected item or request.

  2. Red Cross Button: This button, represented by a red cross, usually signifies a “Waitlist” action. Clicking it would waitlist a selected item or request.

  3. PDF Icon Button: This button, represented by a PDF document icon, is likely used to generate or view a PDF document related to the selected item.

  4. Download Icon Button: This button, represented by a download arrow, is typically used to download a file or data related to the selected item.

  5. Share Icon Button: This button, represented by a share icon, likely allows you to share the selected item or data via different platforms or methods.

INVOICE MANAGEMENT SYSTEM

  1. #: This column is likely the serial number or index of the rows in the table, providing a simple count of the entries.

  2. Invoice No: This column displays the invoice number, a unique identifier for each invoice.

  3. Invoice Date: This column shows the date when the invoice was issued.

  4. Invoice Due Date: This column indicates the date by which the invoice is due to be paid.

  5. Vendor Name: This column lists the name of the vendor or supplier who issued the invoice.

  6. Requisition No: This column shows the requisition number, which is an internal reference number for the purchase request. In this image, it appears to be empty or not used.

  7. Store: This column indicates the store or location associated with the invoice.

  8. Taxable Amount: This column shows the amount on which tax is calculated.

  9. GST Amount: This column lists the amount of Goods and Services Tax (GST) applied to the taxable amount.

  10. Total Amount: This column shows the total amount of the invoice, including taxes.

  11. Phone Number: This column provides the contact phone number of the vendor.

  12. GST Number: This column displays the GST number of the vendor, which is their tax identification number.

  13. Payment Status: This column indicates the status of the payment for the invoice. In the image, it appears to be empty or not used.

  14. Status: This column shows the approval status of the invoice, such as “Pending” or “Approved.” Next to the status, there are action buttons for approving, rejecting, generating a PDF, downloading, or sharing the invoice.

Clicking on any invoice will open its details.

RECURRING INVOICE

Upon creation, the order will be shown with an active status.

Orders can be edited, details viewed, and inactivated as needed.

By clicking Edit:

A pop up will appear.

By clicking “Edit Recurring Invoice”.

"Recurring Invoice Updated Successfully" will appear.

View the details of recurring invoices by clicking on "Detail."

By clicking “Inactive”. Status will show “Active” Option.

To initiate a recurring order, select "Create Recurring Invoice."

Enter the necessary details for the recurring order according to requirements and “Submit”.
