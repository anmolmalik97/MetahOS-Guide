Store Management

“This screen provides a quick overview of your store information, including name, GST license number, address, and code.”

Click the blue button to add a new store.

To create a new store, the user provides the following details:

  • Store Code (for internal identification)
  • Store Name (easily recognizable name for your store)
  • Store License Number
  • GST IN (Goods and Services Tax Identification Number)
  • Address (complete store location)
  • Assigning Authority (the Department or any person or organization authorized)
  • Notes (additional information you want to store about the store)

    From here, users can choose the authority for which they can select the store.

    Users can modify the store details from here by clicking on the yellow edit button.
    Also, there is a button available to enable or disable the store.
