Facility>Inventory Management>Item Manager>Action>Create

Choosing “Create” from the Actions menu brings up this screen. Here, you’ll enter the necessary information to add a new item.


On this screen, you’ll find a “Select Type” dropdown menu. Choose between “Product Order” if you’re adding a new product to the system, or “Work Order” if you’re requesting service or maintenance.

Here, you’ll enter specific information for the item, such as its product name, any associated barcode, and a unique internal ID for your system.

Here, you’ll find the HSN code selection. Choosing the appropriate code will trigger the system to automatically fill the GST percentage in the dedicated GST section.

Note: The HSN code section will only become available for input after you begin entering a number in field.


This section categorize the item by selecting its type. Options include medicine, stationery, patient consumables, supply items, asset etc,

Here user will fill the following sections-

  • Par Level: This is the ideal stock level you want to maintain to avoid stockouts.
  • Critical Par Level: This is the minimum level of stock you should have before placing a new order to avoid stock disruption.
  • Minimum Temperature: This is the lowest temperature the item can safely be stored at.
  • Maximum Temperature: This is the highest temperature the item can safely be stored at.

On this screen, you’ll choose the unit that best describes how the item is measured. This could be tablets, capsules, inhalers, or other relevant options.

In this section, the user will specify the medication’s composition, which refers to the ingredients that make up the medicine. And, they will need to select the generic name, which is the standard name for the active ingredient.

Here, you’ll choose the physical form of the medication. This could be tablets, capsules, syrups, injections, or other available options.

This section lets you choose the medication’s scheduling category. This might refer to regulations on how the medicine can be dispensed or refilled.

In this section, users can classify the item based on the ABC-VED inventory management system. This system considers factors like cost and criticality to determine appropriate inventory control strategies.

This section focuses on pricing the new item. Here, you’ll enter various price points including:

  • MRP (Maximum Retail Price): This is the highest price at which the item can be legally sold to consumers.
  • Selling MRP: This is the actual price your company will sell the item to customers.
  • Outsource Price: If applicable, this is the price you pay to an external supplier to obtain the item.
  • CGHS Price: This refers to the price set by the Central Government Health Scheme (CGHS) in India, if the item is covered by this program.

    This screen appears after selecting “Add Package.” Here, you can specify details about a package associated with the item. If a package isn’t applicable, simply click “Remove” to exclude this section.

    Clicking “Add store” will display a screen for you to fill the information.


    Clicking “Create new item” will add the item to your inventory. You’ll see a confirmation message that says "Inventory item created successfully."
