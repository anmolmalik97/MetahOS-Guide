After the login, the user reaches the landing page

Facility:

Facility>Inventory Management

*To access Inventory Management, navigate to the top menu and select “Facility” followed by "Inventory Management."

Inventory Management:

Once you click “Inventory Management,” the user will be on a page for creating new items. Just click “Item Manager” to get started.

Facility>Inventory Management>Type

In this section, the user can choose the Type of an item, similar to how the user picks a type of medicine.

Facility>Inventory Management>Showing

This section allows users to monitor the par levels set for each inventory item.

Indent:

Facility>Inventory Management>Indent

To initiate an indent request, click the “Indent” button. This will open the indent creation modale.

Stock Expiry:

Facility>Inventory Management>Stock Expiry

This section allows users to filter and view stock expiry dates based on a specific date and a range of subsequent days.

Store:

Facility>Inventory Management>Selecting Store

This section allows the user to select the specific store.

Item Manager:

Facility>Inventory Management>Item Manager

Fill out these details to add a new item to the user inventory.

Create:

Facility>Inventory Management>Item Manager>Action>Create

Clicking “Action” opens the item creation menu. Here, fill out the required details for the new item and then click “Create Item” to finalize its creation.

Inward:

Facility>Inventory Management>Item Manager>Action>Inward

User clicks on Inward from the “Action” menu to Inward a single item.

Note- For Inward any item firstly we have to select any item from search option.

Multi Inward:

Facility>Inventory Management>Item Manager>Action>Multi Inward

User clicks on Multi Inward from the “Action” menu to Inward multiple item.

Stock& Prices:

Facility>Inventory Management>Item Manager>Action>Stock& Prices

From these options user can check the information about the specific products like stock.

Edit:

Facility>Inventory Management>Item Manager>Action>Edit

Here, the user can make changes to product information.

Outward:

Facility>Inventory Management>Item Manager>Action>Outward

  1. Used: Products that have been consumed or utilized in some way.
  2. Wastage: Materials or products that are discarded or lost during production or consumption.
  3. Expired: Products that have reached their expiration date and are no longer suitable for use or sale.
  4. Vendor return: Goods returned to the supplier due to defects, overstock, or other issues.
  5. Reconcile The process of comparing and adjusting records to ensure they accurately reflect the actual inventory levels or financial transactions.

Outward Report:

Facility>Inventory Management>Item Manager>Action>Outward Report

Use the outward report to track outgoing inventory by specifying a date, Store, and Outward Type. We can also download the Outward report by clicking on the green download button.

Edit Invoice:

Facility>Inventory Management>Item Manager>Action>Edit Invoice

The Edit Invoice function provides the ability to modify details on a previously issued invoice, ensuring accuracy and reflecting any necessary changes. This can be done by clicking on the yellow button and can add the reason for the change by clicking on the red button.

NOTE- Only authorized personnel can adjust invoices, and any edits must be done within a 24-hour window.

Transfer:

Facility>Inventory Management>Item Manager>Action>Transfer

The transfer section facilitates the movement of inventory between stores through an indent request process.

Direct Transfer:

Facility>Inventory Management>Item Manager>Action>Direct Transfer

The direct transfer section facilitates the movement of inventory between stores without any indent request process.

Transfer Logs:

Facility>Inventory Management>Item Manager>Action>Transfer Logs

The transfer log tracks the file of purchases, direct transfers, dispensed, and returns on the system and records a lot of basic and additional information.

Transfer Request:

Facility>Inventory Management>Item Manager>Action>Transfer Request

Transfer request is used to move stock or products between different locations, like warehouses or stores.

Transfer Acknowledgement:

Facility>Inventory Management>Item Manager>Action>Transfer Acknowledgement

The transfer acknowledgment is used for the confirmation that the transferred inventory has been received at the designated destination store. It completes the transfer process and ensures all parties involved are informed.
